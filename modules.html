Hackathon

During the course of about a three week period, three classmates and I worked together on our hackathon, which at the time, was a pretty daunting task. Our proposal started with the idea to create a mental health resource, since our group noticed that there weren't a lot of resources available to students online about mental health, as a lot of the main ones that students would try to access at school were blocked by the OCDSB. To connect it to computer science though, we had to find a way to involve mental health in some sort of code or programming. So, we created a website using HTML, Javascript and CSS using Glitch as our web developer. This hackathon was the first time I created a website, so it helped me to get more comfortable with using HTML and CSS, while also learning what I consider a pretty basic skill in web development. Working through different files as a group was another thing I had to work on during this project, because I had never really worked on a larger project in a group before with computer science. Overall, this project helped me grow my skills regarding web development, especially in a fun group setting, and I even got to learn about more mental health resources that I will be able to use or provide to others in the future if they need them.
